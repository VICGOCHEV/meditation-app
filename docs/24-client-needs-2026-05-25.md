### 3. VK Mini App — недостающие ключи

Зайди в `vk.com/editapp?act=manage&id=54600947` → раздел «Настройки».
Пришли два значения:

```
VK_SECURE_KEY    = ____________________
VK_SERVICE_TOKEN = ____________________
```

Без них VK-обёртка не может проверять подпись `vk_signature` —
приложение внутри VK будет показывать ошибку доступа всем кроме тебя.

Там же убедись, что галка **«Открытое API»** — включена.

### 4. Соцсети для входа

В правках 21 мая написано: «сначала логин по почте — или по аккаунту
соц сети». Какие именно соцсети подключаем?

- [ ] **VK ID** — рекомендую, у тебя уже есть VK app
- [ ] **Yandex ID** — рекомендую, для РФ-аудитории нативный вариант
- [ ] **Google** — работает, но многие пользователи РФ без VPN не смогут залогиниться
- [ ] **Apple** — нужен только если будет iOS-обёртка через App Store
- [ ] **Telegram Login** — бонус, удобно для тех, кто пришёл из бота

Мой совет: **VK ID + Yandex ID + email-пароль**. Этого хватит на 95% аудитории.

### 5. HUE утро/вечер — что меняем?

Ты сказал «утро — красное, вечер — зелёное, меняются цвета». Уточни:

- **Вариант A** — меняется **вся цветовая тема приложения** (фон,
  AmorphSphere, акценты) в течение дня. Это большая работа, ~3-4 дня,
  и эффект может выглядеть странно — медитативное приложение должно
  быть нейтральным по цвету.
- **Вариант B** — меняется только **цвет иконки/бейджа push-уведомления**
  и, возможно, обводки конкретной утренней/вечерней практики на главной.
  Это полдня.

Рекомендую **B** — он понятнее по UX и не ломает атмосферу.

  - «Утреннее погружение»
  - «Обращение к себе»

- Женский — полная запись + 10-секундное превью

- Энергия очищения
- Энергия жизни

Каждая — полный трек + 10-секундное превью.

### Telegram-бот — заполнить через @BotFather

- **Name** — короткое имя бота (видно в чате)
- **Description** — текст «о боте» (длинный, показывается при `/start`)
- **About** — короткое (160 символов) для карточки бота
- **Botpic** — аватар (512×512 PNG)
- **Commands list** — список команд, например `/start /menu /help /support`

После заполнения — пришли мне `@username` бота, я свяжу с приложением.
