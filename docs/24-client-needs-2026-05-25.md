## 🔴 БЛОКЕРЫ — ждём твоего решения

### A. SMTP через Yandex 360

Для отправки писем приложения (восстановление пароля, чеки донейшна, форма обратной связи).

**Шаг 1**: зайти на `https://360.yandex.ru/business` → подключить тариф **«Стандартный»** (~339-450 ₽/мес за 1 пользователя). 30 дней триал бесплатно — можно стартовать без оплаты.

**Шаг 2**: в админ-панели Yandex 360 → «Домены» → «Добавить домен» → ввести `all-relaxme.ru`.

**Шаг 3**: Yandex покажет 5 DNS-записей для добавления у регистратора домена:

| Тип | Имя | Значение | Что |
|---|---|---|---|
| TXT | `@` | `yandex-verification: ABC123...` | подтверждение владения (потом удалить) |
| MX | `@` | приоритет 10, `mx.yandex.net.` | куда приходит почта |
| TXT | `@` | `v=spf1 redirect=_spf.yandex.net` | кто может слать от имени домена |
| TXT | `mail._domainkey` | `v=DKIM1; k=rsa; t=s; p=...` | подпись писем |
| TXT | `_dmarc` | `v=DMARC1; p=none; rua=mailto:postmaster@all-relaxme.ru` | политика |

Точные значения — увидишь в Yandex 360 при добавлении домена.

**Шаг 4**: создать ящик `noreply@all-relaxme.ru` → Настройки → «Доступ к ящику» → «Пароли приложений» → создать пароль с галкой **«IMAP / SMTP»** → скопировать сгенерированный пароль (показывается один раз!).

**Шаг 5**: прислать мне:
```
SMTP_HOST  = smtp.yandex.ru
SMTP_PORT  = 465
SMTP_USER  = noreply@all-relaxme.ru
SMTP_PASS  = <app-password>  (длинная строка)
```

### C. Соцсети для логина

Какие провайдеры подключаем?

- [ ] **VK ID** — рекомендую (у тебя уже есть VK app)
- [ ] **Yandex ID** — рекомендую (для РФ-аудитории нативный)
- [ ] **Google** — работает, но в РФ часто без VPN не залогиниться
- [ ] **Apple** — только если будет iOS-обёртка через App Store

Мой совет: **VK ID + Yandex ID + email/пароль**. Этого хватит 95% аудитории.
