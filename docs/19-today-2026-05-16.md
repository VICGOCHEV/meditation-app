## 2. Backend — Fastify + Prisma + Postgres

**Путь:** `/opt/meditation-api` на том же VPS, порт `127.0.0.1:3001`, systemd unit.

Маршруты:
- `POST /api/auth/register` — email-регистрация, сразу отдаёт JWT (без второго round-trip).
- `POST /api/auth/login` — email + пароль → JWT.
- `POST /api/auth/verify` — 501, ждёт SMS-провайдера.
- `POST /api/auth/reset` — silent ok (anti-enumeration).
- `GET /api/auth/me` — текущий пользователь.
- `GET /api/progress` — снапшот всего состояния (subscription, unlocked, completed, tracker, KT history, bonus).
- `POST /api/practices/:id/complete` — отметка прохождения + tracker-day одним вызовом.
- `POST /api/checkin` — IS чекин (`q1..q4`).
- `POST /api/deep-analysis` — KT + автооткрытие следующей `awareness` + проверка бонуса.
- `POST /api/subscription` — активация / продление на 30 дней + автооткрытие `a1`.
- `DELETE /api/subscription` — мягкая отмена (флаг active=false, expiresAt сохраняется).
- `GET /api/health`.

Prisma-схема: 8 моделей (`User`, `Subscription`, `UnlockedAwareness`, `PracticeCompletion`, `TrackerDay`, `Checkin`, `KtEntry`, `BonusUnlock`).

Бизнес-логика (бонус, KT-формула) вынесена в `utils/` и зеркалит фронт; источник истины — сервер.

Лог: [17-backend-implementation-log.md](17-backend-implementation-log.md).

## 4. AmorphSphere на плеере — финал визуала

Долгая итерация с обратной связью пользователя:
- Шейдер перекомпонован: убран `mix-blend-mode: screen` (он постоянно ловился стэкинг-контекстом framer-motion и давал чёрные части).
- WebGL clear через `setClearColor(0,0)` + `setClearAlpha(0)`.
- Альфа канала привязана к яркости пикселя (`smoothstep` от `lum`) — края растворяются в прозрачность вместо чёрного.
- Добавлен ambient bias `cDeep * body * 0.25` чтобы вернуть теневую (нижне-левую) часть, которая иначе исчезала.
- Smoky-центр восстановлен через `mix(0.55, 1.0, cut)` — внутренние пересечения "лоскутов" опять полупрозрачные.
- AudioPlayer оборачивает сферу в `AnimatePresence` с fade-in 0.9s (только после первого `onFirstPaint`) и fade-out 0.28s.
- Player.jsx перехватывает все back-навигации через `exit(target)` — даёт сфере 280мс на исчезновение, чтобы блендинг не схватывался роутом.

## 6. Bug-fixes из аудита

- **`bonusProgress is not a function`** — пустой экран после регистрации. Сервер возвращал поле `bonusProgress` (объект), а в сторе метод с тем же именем; `set(p)` затирал метод. Метод переименован в `getBonusProgress()`.
- **DeepAnalysis/index.jsx** — удалены мёртвые селекторы `unlockNext` / `unlockBonus` (теперь это делает сервер внутри `recordDeepAnalysis`).
- **/api/health 404** через Caddy — `handle_path /api/*` срезал префикс, переключили на `handle /api*`.
- **Frontend VITE_API_URL** оставался `http://localhost:3000/api` от мок-стека — поправлен на `/api`.
