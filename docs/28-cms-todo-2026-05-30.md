## 🧩 Бэклог полировки (необязательно для запуска)

- [ ] **Drag-сортировка голосов/музыки** — бэк-ручки reorder готовы
      (`/admin/voices/reorder`, `/admin/music/reorder`), но в SPA
      `AudioCollection` пока без DnD (грип нарисован, перетаскивание не
      подключено). Для 2 голосов / 3 треков порядок и так задаётся `order`.
- [ ] **Автосохранение черновика** — сейчас явные «Сохранить»/«Опубликовать».
      В спеке было «nice-to-have».
- [ ] **Пагинация юзеров в UI** — бэк отдаёт `page/pageSize`, SPA грузит
      первые 50. Добавить кнопки страниц при росте базы.
- [ ] **Страница медиа-библиотеки** — бэк `GET /admin/media` есть, отдельного
      экрана в SPA нет (загрузка идёт из ячеек/слотов, библиотека не нужна на старте).
- [ ] **Адаптив под планшет** — базовый есть (сайдбар прячется < md), точечно

## 🗒️ Открытые вопросы (решить при выкате)
1. `PracticeCompletion.practiceId` на проде = Strapi `documentId`. Скрипт
   миграции кладёт его в `slug` → совпадает. **Проверить на реальных данных**,
   что у текущих юзеров прогресс не слетел.
2. Роль первого админа: `admin` (может выдавать подписки) или `editor`.
3. Лимит размера аудио `MAX_AUDIO_BYTES` (сейчас 60 МБ) — хватит ли для
   20-мин дорожек в хорошем битрейте.
