## Как добавить практику

1. Слева кликни **Practice** → справа сверху синяя кнопка **Create new entry**.
2. Заполни поля:
   - **title** — название как видит пользователь (например, «Дыхание 4-7-8»).
   - **block** — выбери из выпадашки:
     - `relaxation` — бесплатный блок «Расслабление»
     - `awareness` — платный блок «Осознанность» (открывается по подписке)
     - `author` — «Авторский» блок (бонус или покупка)
   - **audio** — обязательное поле. Кликни «Click to add an asset» → перетащи mp3 → нажми **Upload**.
   - **duration_sec** — НЕ трогай. Поле заполнится автоматически после загрузки аудио (в секундах).
   - **description** — необязательно, короткое описание для админки.
   - **price** — только для блока `author`, если хочешь продавать поштучно. Сумма в рублях, например `290`.
   - **order** — необязательно, число для сортировки. Чем меньше — тем выше в списке. По умолчанию 0.
   - **month_slot** — пока пропусти, это для будущей ежемесячной ротации.
3. Справа сверху кнопка **Save** (просто сохранить как черновик) и **Publish** (опубликовать).

   ⚠️ **Запись попадёт в приложение только после Publish.** Без публикации она лежит как черновик и приложение её не видит.

4. После Publish — открой главную приложения http://188.137.177.136/ и убедись, что карточка появилась в нужном блоке.

## Как добавить голос проводника

Создаются 2 голоса один раз: мужской и женский. Можно заменить аудио в любой момент.

1. **Voice** → **Create new entry**.
2. Заполни:
   - **name** — что увидит пользователь («Мужской» / «Женский»).
   - **code** — техническая метка, по которой приложение различает голоса:
     - для мужского пиши строго `male`
     - для женского — `female`
   - **audio_full** — полная дорожка голоса (mp3).
   - **audio_preview** — обязательно короткое превью 5–10 секунд (это то, что воспроизводится на онбординге, когда юзер пробует голос).
   - **active** — оставь `true`, если голос доступен.
   - **order** — порядок (0 у мужского, 1 у женского — или как хочется).
3. **Publish**.

Чтобы поменять аудио позже — открой существующую запись, кликни на файл, нажми крестик, перетащи новый, **Save** + **Publish**.

## Как добавить фоновую музыку

Должно быть 3 трека (по ТЗ).

1. **Music track** → **Create new entry**.
2. Заполни:
   - **title** — название («Спокойствие», «Природа», «Космос» или твоё).
   - **audio_full** — полная дорожка.
   - **audio_preview** — короткое превью 5–10 сек.
   - **active** — `true`.
   - **order** — 0, 1, 2 — порядок в выборе на онбординге.
3. **Publish**.

## Загрузка аудиофайлов

- Формат: **MP3** (предпочтительно) или AAC.
- Битрейт: 128–192 kbps для практик и голоса; 96–128 kbps для фоновой музыки.
- Размер: не критично, главное чтобы влез в сервер. 50–100 MB на файл норма.
- Имена файлов: лучше латиницей без пробелов (`relax_1.mp3`), но Strapi принимает и кириллицу.

Все загруженные файлы хранятся в разделе **Media Library** (картинка-папка в левой панели иконок). Можно посмотреть всё что загружалось, переименовать, удалить (но удалять то, на что ссылается опубликованная практика, не стоит — приложение поломается).

## Если что-то пошло не так

- **Не загружается аудио** — проверь формат (mp3/aac) и размер. Если файл > 100 MB, скажи нам, поднимем лимит.
- **duration_sec не заполнился** — сохрани запись ещё раз. Если упорно не заполняется, скажи нам — проверим хук.
- **Запись опубликована, но в приложении не видна** — подожди 1–2 минуты. Если не появилось — пиши нам.
- **Случайно удалила запись** — не страшно, восстановим через бэкап БД.
