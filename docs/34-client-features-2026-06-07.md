# Meditation app — что это даёт

## Приложение

- Быстрое одностраничное приложение без перезагрузок, ощущается как нативное
- Цвет всего приложения сам меняется в течение дня — утро тёплое, вечер зелёное
- Пользователь может зафиксировать любимое настроение вручную
- Медитативная анимированная атмосфера вместо плоского экрана
- Заставка загрузки в цвет времени суток, без цветового скачка
- Слушатель сам собирает практику — чей голос и какая подложка
- Можно послушать диктора до выбора, а не вслепую
- Приложение оценивает реальное состояние пользователя
- Замеры прогресса в ключевых точках курса, ощущение динамики
- Контролируемый темп, удерживает привычку
- Пользователь сам управляет напоминаниями и их временем
- Полное удаление аккаунта и данных по требованию
- Эффект сообщества — «ты не один»
- Канал для отзывов и вопросов прямо из приложения
- Оплата подписки не уводит со страницы, не ломается

## Тарифы

- Три уровня подписки + поштучные авторские
- Понятная картина, что доступно пользователю
- 7 дней бесплатного триала снижают барьер первой покупки
- Подкаст не путается с платным контентом

## Аккаунт и данные

- Безопасный вход вместо хранения в браузере
- Прогресс не теряется при смене устройства или чистке браузера
- Корректная логика подписок без потери оплаченного доступа
- Автоматическая активация подписки после оплаты
- Обновления без простоя
- Фирменные красивые письма вместо голого текста
- Письма не теряются даже при заблокированном провайдере

## Уведомления

- Вход через Telegram и ВКонтакте без отдельной регистрации
- Бот и пуши работают с российского сервера
- Персональные напоминания в нужное местное время
- Разнообразные тексты напоминаний, не спам

## Управление контентом (CMS)

- Удобная админка в дизайне приложения
- Загрузка аудио осмысленно, без путаницы в полях
- Масштаб с 1 до 20 практик без программиста
- Длительность трека считается сама
- Замена практик каждый месяц без правки кода
- Статистика на виду + выдача доступа вручную
- Прогресс пользователей не теряется при переезде
- Все отзывы и вопросы в одном месте

## Промо

- Кинематографичная промо-страница, продающая приложение

## Инфраструктура

- Рабочий сайт с надёжным HTTPS
- Аналитика: онбординг, практики, подписки, отмены
- Предсказуемое обновление без даунтайма
